# Keyboard Shortcuts

Quarry is designed for keyboard-driven productivity. This reference documents all available keyboard shortcuts organized by menu.

## Quick Reference

| Action            | Shortcut                                   |
| ----------------- | ------------------------------------------ |
| Run Query         | <kbd>⌘</kbd> + <kbd>R</kbd>                |
| Explain Query     | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>E</kbd> |
| Open Database     | <kbd>⌘</kbd> + <kbd>O</kbd>                |
| Search All Tables | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>F</kbd> |
| Diagram           | <kbd>⌘</kbd> + <kbd>D</kbd>                |

## File

| Action                  | Shortcut                                   |
| ----------------------- | ------------------------------------------ |
| New Database            | <kbd>⌘</kbd> + <kbd>N</kbd>                |
| Open Database           | <kbd>⌘</kbd> + <kbd>O</kbd>                |
| Open Encrypted Database | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>O</kbd> |
| Connect to Server       | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>K</kbd> |
| Close Database          | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>W</kbd> |

The File menu also includes **Create Demo Database…**, **Save Connection Profile…**, and the CSV/JSON import and CSV/JSON/SQL export commands (no default shortcuts).

## Query

| Action        | Shortcut                                   |
| ------------- | ------------------------------------------ |
| Run Query     | <kbd>⌘</kbd> + <kbd>R</kbd>                |
| Explain Query | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>E</kbd> |
| Save Query    | <kbd>⌘</kbd> + <kbd>S</kbd>                |

## Database

| Action            | Shortcut                                   |
| ----------------- | ------------------------------------------ |
| Search All Tables | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>F</kbd> |
| Diagram           | <kbd>⌘</kbd> + <kbd>D</kbd>                |
| Compare           | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>D</kbd> |
| Refresh Table     | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>R</kbd> |

The Database menu also includes **Maintenance**, **Generate Mock Data…**, **Save Schema Snapshot…**, **Compare Schema Snapshot…**, **Backup Database…**, and **Restore Database…** (no default shortcuts).

## Text Editing

Standard macOS text-editing shortcuts (<kbd>⌘</kbd> + <kbd>C</kbd>, <kbd>⌘</kbd> + <kbd>V</kbd>, <kbd>⌘</kbd> + <kbd>Z</kbd>, <kbd>⌘</kbd> + <kbd>A</kbd>, …) work throughout the app, including the SQL editor.

## Troubleshooting

### Shortcuts Not Working

If keyboard shortcuts aren't responding:

1. **Check focus** - Make sure the relevant window is frontmost
2. **Check for conflicts** - System-wide macOS shortcuts may override app shortcuts (see System Settings → Keyboard → Keyboard Shortcuts)
3. **Check the menus** - Every shortcut is shown next to its menu item, so the menu bar is the authoritative reference
