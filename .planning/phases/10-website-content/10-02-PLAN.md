Modify the features array and add size property:

```tsx

  { key: 'crossPlatform', size: 'tall', icon: '...', ... },
  { key: 'visualization', size: 'default', icon: '...', ... },
```

Update the JSX to use bento-grid class and apply size-based classes:

```tsx
<div className="bento-grid">
  {features.map((feature) => (
    <div key={feature.key} className={`bento-item bento-${feature.size}`}>
      {/* icon, title, description */}
    </div>
  ))}
</div>
```

2. **Update Features.css**:

Add bento grid layout (after existing styles or replacing .features-grid):

```css
.bento-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: 160px;
  gap: 16px;
  max-width: 1200px;
  margin: 0 auto;
}

.bento-item {
  background: var(--card);
  border: 1px solid var(--border);
  border-radius: var(--border-radius-xl);
  padding: 24px;
  display: flex;
  flex-direction: column;
  transition: border-color 0.2s ease;
}

.bento-item:hover {
  border-color: var(--primary);
}

/* Size variants */
.bento-large {
  grid-column: span 2;
  grid-row: span 2;
}

.bento-wide {
  grid-column: span 2;
}

.bento-tall {
  grid-row: span 2;
}

.bento-default {
  /* 1x1, no span needed */
}

/* Responsive: collapse to 2 columns on tablet, 1 on mobile */
@media (max-width: 1024px) {
  .bento-grid {
    grid-template-columns: repeat(2, 1fr);
  }
  .bento-large {
    grid-column: span 2;
    grid-row: span 1;
  }
  .bento-tall {
    grid-row: span 1;
  }
}

@media (max-width: 640px) {
  .bento-grid {
    grid-template-columns: 1fr;
  }
  .bento-large,
  .bento-wide {
    grid-column: span 1;
  }
}
```
