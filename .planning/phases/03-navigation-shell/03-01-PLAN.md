## Objective

Transform all three tab bar components from underline-based active indicators to pill/background-style active states, aligning with Linear/Raycast visual patterns.

## Context

Current state analysis:

- **ConnectionTabBar** (L275-283): Uses `borderBottomWidth: '2px'` + `borderBottomColor: connectionColor`
- **DataTabBar** (L155-157): Uses `border-b-gold border-b-2` for active state
- **QueryTabBar** (L110-112): Uses `shadow-[inset_0_-2px_0_0_var(--color-primary)]` for underline effect

Target: Background pill with subtle rounded corners, no underline indicators.

**Update ConnectionTabBar active state styling**



5. Remove the `border-r` class from individual tabs to reduce visual noise
   </task>
