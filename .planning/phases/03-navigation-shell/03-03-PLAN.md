## Objective

Flatten the Sidebar visual hierarchy by reducing nesting decoration, removing heavy borders, and simplifying section headers. The goal is a cleaner, Linear-style sidebar that feels less cluttered while maintaining functionality.

## Context

Current state analysis (Sidebar.tsx):

- Uses `border-l` vertical lines for nested sections (L1174, L1195, L1258, L1322)
- Section headers use ChevronDown/ChevronRight icons with UPPERCASE text styling
- Heavy indentation: `ml-3 border-l pl-2` for schema content, `ml-1.5 border-l pl-2` for table/view lists
- Multiple levels: Schema → Section (Tables/Views/Triggers) → Items

**IMPORTANT**: This is a large, complex component (62KB). Changes must be conservative to avoid breaking functionality. Focus only on visual styling, not structural changes.

Target: Reduced visual nesting while maintaining collapsible sections.

## Tasks

<task id="1">
**Remove nested border-left lines for single-schema databases**

When there's only one schema (most common case), remove the visual nesting lines:

In the SchemaSection component (around line 1172-1175):

```tsx
// Before:
className={
  showSchemaHeader ? 'border-border/40 ml-3 border-l pl-2' : ''
}

// After:

  showSchemaHeader ? 'ml-2' : ''
}
```

For the table/view/trigger lists (lines 1195, 1258, 1322):

```tsx

// After:

```

This removes the vertical connector lines, creating a flatter appearance.
</task>

<task id="2">
**Simplify section header styling**

Update the Tables/Views/Triggers section button styling (lines 1182, 1245, 1309):

```tsx
// Before:
className =
  'text-muted-foreground/80 hover:text-muted-foreground hover:bg-accent/30 flex w-full items-center gap-1 rounded-sm px-1 py-0.5 text-[0.75em] font-medium tracking-wider uppercase';

// After:
className =
  'text-muted-foreground hover:text-foreground hover:bg-muted/50 flex w-full items-center gap-1.5 rounded-md px-2 py-1 text-xs font-medium';
```

Changes:

- Remove `uppercase` and `tracking-wider` for less aggressive styling
- Increase padding for better touch targets (`px-2 py-1`)
- Use `text-muted-foreground` (not `/80` opacity) for consistency
- Change `rounded-sm` to `rounded-md` for softer appearance
- Bump gap to `gap-1.5` for better icon spacing
  </task>
