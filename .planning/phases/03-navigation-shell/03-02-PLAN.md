## Objective

Simplify the Activity Bar styling to a minimal, flat Linear/Raycast-style appearance. Remove heavy gradients, glow effects, and decorative elements while maintaining functionality.

## Context

Current state analysis (ActivityBar.tsx):

- Already icon-only design (good ✓)
- Uses complex gradient backgrounds: `from-primary/20 via-primary/12 to-primary/5 bg-gradient-to-br`
- Heavy shadows with glow: `shadow-[0_2px_8px_rgba(16,185,129,0.2),inset_0_1px_0_rgba(255,255,255,0.1)]`
- Drop shadow on icons: `drop-shadow-[0_0_6px_rgba(16,185,129,0.5)]`
- Active indicator uses gradient bar: `from-primary via-primary bg-gradient-to-b to-[var(--accent-cyan)]`
- Container has gradient background and decorative elements (noise overlay, gradient lines)

Target: Flat, minimal styling with simple background highlight for active state.
