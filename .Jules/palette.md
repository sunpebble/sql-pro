## 2024-05-19 - Added aria-labels to close and copy buttons\n**Learning:** Icon-only buttons without `aria-label` are a common accessibility issue. For close buttons (e.g. `X` icon) and copy buttons, providing an accessible name via `aria-label` using the i18n `t` function is essential for screen reader support.\n**Action:** Ensure all icon-only buttons have an `aria-label`, preferably localized.
